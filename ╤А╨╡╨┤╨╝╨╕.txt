# Wildberries API Project

## ?? Что это
Проект для работы с Wildberries API: карточки товаров, заказы, продажи, отчёты CTR/STR.

Структура:


wb_project/
+-- wb_api.py # Работа с API WB
+-- reports.py # Табличные отчёты (CTR/STR)
+-- analyzer.py # Аналитика и графики
+-- config.py # Настройки (токен, заголовки)
+-- main.py # Точка входа
L-- README.md # Инструкция


---

## ?? Быстрый старт

1. Установи зависимости:
```bash
pip install requests tabulate matplotlib


Вставь свой API-токен в файл config.py:

TOKEN = "ВАШ_ТОКЕН"


Запусти проект:

python main.py

?? Возможности

wb_api.py

get_cards() > список карточек

get_orders() > заказы

get_sales() > продажи/выкупы

get_stocks() > остатки

reports.py

Строит таблицу CTR/STR по карточкам.

analyzer.py

Рисует график продаж по дням.

main.py

Упрощённое меню (отчёт или график).

?? Что уже сделано

Получение карточек, заказов, продаж.

Таблица CTR/STR в консоли.

График продаж (matplotlib).
